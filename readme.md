# Requisitos funcionais

[x] Deve ser possível criar um usuário
[x] Deve ser possível logar com o usuário
[x] Deve ser possível deslogar com o usuário
[x] Deve ser possível registrar uma refeição, com nome, descrição, data, hora e se está dentro da dieta
[ ] Deve ser possível editar a refeição, alterando todos os dados
[ ] Deve ser possível apagar uma refeição
[x] Deve ser possível listar todas as refeições de um usuário
[x] Deve ser possível visualizar uma única refeição
[x] Deve ser possível recuperar as métricas:
    [x] Quantidade total de refeições registradas
    [x] Quantidade total de refeições dentro da dieta
    [x] Quantidade total de refeições fora da dieta
    [x] Maior sequência de refeições dentro da dieta

# Regras de negócio

[x] Para realizar as operações, o usuário deve estar logado
[x] As operações devem ser vinculadas ao usuário, e não deve ser permitido interagir com operações de outro usuário
